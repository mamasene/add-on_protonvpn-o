Accessibilité ProtonVPN pour NVDA

Auteur : Mama Sene <tech.access33@gmail.com>
Compatibilité NVDA : 2025.1 et ultérieur

Présentation

Cette extension améliore l'accessibilité de l'application ProtonVPN pour Windows avec NVDA.

Elle fournit :

  • Un étiquetage correct des éléments de l'interface
  • Des informations vocales sur l'état du VPN (adresse IP, pays, fournisseur, trafic)
  • Des raccourcis clavier pour les actions VPN courantes

Utilisation

Une fois l'extension installée, ouvrez simplement ProtonVPN.

L'extension améliore automatiquement :

  • L'étiquetage des boutons et des contrôles
  • Les annonces d'état de connexion
  • Les informations de trafic et réseau

Raccourcis clavier

  • Contrôle+Maj+D : Connecter / Déconnecter le VPN
  • Contrôle+Maj+K : Activer / Désactiver le Kill Switch
  • Contrôle+Maj+C : Ouvrir le sélecteur de pays
  • Contrôle+Maj+T : Annoncer les informations de trafic

Remarques

Les messages vocaux (adresse IP, pays, fournisseur, trafic) sont dynamiques. Ils dépendent de l'état actuel de la connexion VPN et des données disponibles.

Cette extension n'envoie aucune donnée et fonctionne entièrement en local.

Il s'agit d'une extension communautaire non affiliée à Proton AG.
